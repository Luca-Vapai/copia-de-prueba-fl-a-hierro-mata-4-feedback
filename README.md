# Copia de Prueba FL - A hierro mata 4 · Feedback site

Review portal for **Copia de Prueba FL - A hierro mata 4**. Pieces and versions are listed at
the root of this site; click into any piece to leave timecoded feedback.

This site is regenerated by the [feedback-loop](https://github.com/Luca-Vapai/feedback-loop)
tool on every render — `config.json` and `transcripts/` get updated as new
versions get published.

Hosted on GitHub Pages. URL: `https://luca-vapai.github.io/copia-de-prueba-fl-a-hierro-mata-4-feedback/`.
